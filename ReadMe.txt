PIC32MX I2S sample code for Pinguino gcc

This code is work on PIC32MX220F032B.

Base on this repository.

https://github.com/uncle-yong/sk1632-i2s-dma-intro

Also very important page PIC32MX I2C know how.

https://tutorial.cytron.io/2017/09/01/i2s-pic32mxmz-direct-memory-access-dma/

I2S connection

MCLK - RA4(PPS)
LRCLK - RB4(PPS)
SDATA - RA1(PPS)
BCLK - RB14

I2S is 48KHz
